Lines

Type: property


Visibility: public


Member Of: TFitsHeader


property Lines[Index: Integer]: string read GetLines write SetLines;


Description:


List of all lines of header, including END and blank lines after END.


  • LineCapacity - number of lines of header, including empty lines after END;
  • LineCount - number of significant lines of header, including END;


For access in to items of line <keyword-separator-value-separator-note> use a TLineBuilder.


Numbering of lines a header begins with zero.


Code Sample:


var

 Fit: TFitsHeader;

 I: Integer;

 Line: string;

begin

 // open file for read and write

 Fit := TFitsHeader.CreateJoin(TFileStream.Create('some.fit', cFileReadWrite));

 // list of lines

 for I := 0 to Fit.LineCount - 1 do

 begin

   Line := Fit.Lines[I];

   // ... some code

 end;

 Fit.Stream.Free;

 Fit.Free;

end;

property Palette: PPalette read FPalette write SetPalette;


Description:


The palette of the graphic image of data. The graphic image of data 8-bit, i.e. 256 colors.


DeLaFits (unit DeLaFitsGraphics) defines the following constants for the palette:


cPaletteGrayScale

     (by default)

cPaletteHot

cPaletteCool

cPaletteBonnet

cPaletteJet


The user can define his palette. To do this, create your own array of TPalette.


Code Sample:


var

 Fit: TFitsBitmap;

 Rgn: TRgn;

 Bmp: TBitmap;


begin


 // open file for read

 Fit := TFitsBitmap.CreateJoin(TFileStream.Create('some.fit', cFileRead));


 // create bitmap

 Bmp := TBitmap.Create;

 

 // define region

 Rgn.X1     := 34;

 Rgn.Y1     := 929;

 Rgn.Width  := 200;

 Rgn.Height := 200;


 // default

 Fit.BitmapRead(Bmp, Rgn);

 Bmp.SaveToFile('pal-default.bmp');


 // set inverse GrayScale palette

 Fit.GraphicColor.Palette      := cPaletteGrayScale;

 Fit.GraphicColor.PaletteOrder := palInverse;

 Bmp.SaveToFile('pal-grayscale-inverse.bmp');

 

 // set direct Jet palette

 Fit.GraphicColor.Palette      := cPaletteJet;

 Fit.GraphicColor.PaletteOrder := palDirect;

 Bmp.SaveToFile('pal-jet-direct.bmp');


 // free

 Bmp.Free;

 Fit.Stream.Free;

 Fit.Free;


end;


Output


pal-default.bmp

pal-grayscale-inverse.bmp

pal-jet-direct.bmp

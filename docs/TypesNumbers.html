Description:


Synonyms of the standard numerical data types. Types are used in methods of read/write data, and also for conversion of standard types to the FITS types, and vice versa.
Logic of naming types: symbol "T" + data size (byte count) + representation of the number.


Representation of numbers (format):

  • f - floating-point;
  • u - unsigned binary integer;
  • c - two's complement binary integer;

TRgn

Types ››

Unit: DeLaFitsCommon


Type: type


PRgn = ^TRgn;

TRgn = record

 X1, Y1: Integer;

 case Integer of

   0: (ColsCount, RowsCount: Integer);

   1: (Width, Height: Integer);

end;


Description:


DeLaFits support the access to the data only for 2D-files, so-called Single Image FITS. The user gets access to the data for a specific region. TRgn this is a record that uniquely identifies a such region.


X1, Y1: Integer;


Coordinates of the Top Left pixel of the region;


case Integer of

 0: (ColsCount, RowsCount: Integer);

 1: (Width, Height: Integer);


The size of the region: the number of columns/rows, or the width/height.

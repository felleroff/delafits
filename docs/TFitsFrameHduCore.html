property HduCore: THduCore2d read GetHduCore2d;


Description:


The 2D-core of the frame header (NAXIS = 2).


Describes the basic properties of the frame: the data block size, bit depth, the coefficients of physical interpretation of the data. See a detailed description of Keywords defined in the FITS Standard: SIMPLE, BITPIX, NAXIS, NAXIS1, NAXIS2, BSCALE and BZERO.

BITPIX  =                   16 / number of bits per data value

BSCALE  =                  1.0 / data scaling factor                            

BZERO   =                  0.0 / data offset

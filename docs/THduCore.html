THduCore

Types ››

Unit: DeLaFitsCommon


Type: type


THduCore = record

 Simple: Boolean;

 BitPix: TBitPix;

 NAxis: Word;

 NAxisn: TA16u1;

 BScale: Double;

 BZero: Double;

end;


Description:


Main meta information a file: values of basic keywords of header. These values describe data: physical representation, dimensionality, bit depth. Basic keywords: SIMPLE, BITPIX, NAXIS, NAXISn, BSCALE and BZERO. See a detailed description of Keywords defined in the FITS Standard.


Simple: Boolean;


Does file conform to the Standard? True - yes, False - no.
DeLaFits support access to the data only for Simple files;


BitPix: TBitPix;


Bits per data value, see TBitPix;


NAxis: Word;


Number of axes. The value field shall contain a non-negative integer no greater than 999, representing the number of axes in the associated data array;


NAxisn: TA16u1;


Dimensional array, size of the axis. The value field of this indexed keyword shall contain a non-negative integer, representing the number of elements along axis n of a data array;


BScale, BZero: Double;


Linear factor and zero point in scaling equation: physical_value = BZERO + BSCALE * array_value.

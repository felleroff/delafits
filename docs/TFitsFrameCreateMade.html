constructor CreateMade(AStream: TStream; const AHduCore: THduCore2d);


Description:


Creates an object and an new FITS-stream.

DeLaFits supports only Single Image FITS: one header and one data unit.


  • AStream − an initialized FITS-stream (new). You create and destroy the Stream-object yourself;
  • AHduCore - the core of header (two-dimensional) in the new FITS-stream, see THduCore2d;


Code Sample:


var

 AStream: TFileStream;

 Fit: TFitsFrame;

 Hdu: THduCore2d;

begin

 AStream := TFileStream.Create('some.fit', fmCreate);

 // prepare a header core

 Hdu.Simple := True;

 Hdu.BitPix := bi16c;

 Hdu.NAxis1 := 1000;

 Hdu.NAxis2 := 2000;

 Hdu.BScale := 1.0;

 Hdu.BZero  := 0.0;

 // create new file

 Fit := TFitsFrame.CreateMade(AStream, Hdu);

 // ... some code

 // free

 Fit.Free;

 AStream.Free;

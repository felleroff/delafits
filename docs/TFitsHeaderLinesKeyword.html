Description:


An list of all the keywords in the header of frame. List contains all the keywords of the frame, including END and the following blank lines.


  • LineCapacity - number of lines of header, including empty lines after END;
  • LineCount - number of significant lines of header, including END;


Numbering of lines a header begins with zero.


Code Sample:


var

 Fit: TFitsHeader;

 I: Integer;

 Keys: string;

begin

 // open file for read

 Fit := TFitsHeader.CreateJoin(TFileStream.Create('some.fit', cFileRead));

 // list of keywords

 Keys := '';

 for I := 0 to Fit.LineCount - 1 do

   Keys := Keys + Fit.LinesKeyword[I] + #13;

 Fit.Stream.Free;

 Fit.Free;

end;

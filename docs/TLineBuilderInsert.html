procedure Insert(Index: Integer; const Item: TLineItem; Cast: Integer);


Description:


Method inserts a new line in position Index.


  • Item - the items a line <keyword-separator-value-separator-note>;
  • Cast - the rule of build a line;


Numbering of lines a header begins with zero


Code Sample:


var

 Fit: TFitsHeader;

 Item1: TLineItem;

 Item2: TLineItem;

begin

 // open file for read and write

 Fit := TFitsHeader.CreateJoin(TFileStream.Create('some.fit', cFileReadWrite));

 // build items:

 // -- as string and Integer

 Item1.Keyword       := 'SOMEKEY1';

 Item1.ValueIndicate := True;

 Item1.Value.Str     := '100';

 Item1.NoteIndicate  := True;

 Item1.Note          := 'my string value';

 // -- as Integer

 Item2.Keyword       := 'SOMEKEY2';

 Item2.ValueIndicate := True;

 Item2.Value.Int     := 100;

 Item2.NoteIndicate  := True;

 Item2.Note          := 'my integer value';

 // insert as string and Integer

 Fit.LineBuilder.Insert(10, Item1, cCastString);

 Fit.LineBuilder.Insert(11, Item2, cCastInteger);

 // free

 Fit.Stream.Free;

 Fit.Free;

end;

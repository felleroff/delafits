constructor CreateMade(AStream: TStream; const AHduCore: THduCore);


Description:


Creates an object as new FITS-stream: writing to the stream TStream; creating the header; creating an array of keywords FHduKeywords; creating the data. The size of the new stream is equal to the minimum size of the header (2880 bytes) plus the size of the data.


  • AStream − an initialized FITS-stream (new). You create and destroy the Stream-object yourself;
  • AHduCore − the core of header of frame , determines the size and format of the data;


Code Sample:


var

 AStream: TFileStream;

 AHduCore: THduCore

 Fit: TFitsHeader;

begin

 AStream := TFileStream.Create('some.fit', fmCreate);

 AHduCore := ToHduCore(True, bi16c, [1024, 1024]);

 Fit := TFitsHeader.CreateMade(AStream, AHduCore);

 // ... some code

 Fit.Free;

 AStream.Free;

end;

Unit: DeLaFitsString


Type: function


function StriToBitPix(const Value: string): TBitPix;


Description:


The function converts String into TBitPix, by the standard. Value has to be equal to one of values:'-64','-32', '8', '16', '32', '64', in another case the function generates an exception 2003:ERROR_STRING_BITPIX.


Code Sample:


var

 B1, B2, B3: TBitPix;

begin

 B1 := StriToBitPix('-32');

 B2 := StriToBitPix('32');

 B3 := StriToBitPix('Abcd');

end;


Output


B1 = bi32f

B2 = bi32c

B3 = biUnknown, EFitsStringException, ERROR_STRING_BITPIX:1003

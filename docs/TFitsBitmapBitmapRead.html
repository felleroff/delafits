procedure BitmapRead(Bitmap: TBitmap); overload;


procedure BitmapRead(Bitmap: TBitmap; const Rgn: TRgn); overload;


Description:


The method creates a BMP-image for the specified region of the graphic image of the data.


  • Bitmap - TBitmap object;
  • Rgn - the region of the graphic image of the data;


The coordinates of the region are specified in the coordinate system of the scene.

If you do not specify the region (overload version) - will be created BMP-image for the full region of the graphic image of the data.


Code Sample:


var

 Fit: TFitsBitmap;

 Rgn: TRgn;

 Bmp: TBitmap;


begin


 // open file for read

 Fit := TFitsBitmap.CreateJoin(TFileStream.Create('some.fit', cFileRead));


 // create TBitmap

 Bmp := TBitmap.Create;


 // define region for the BMP-image of the data

 Rgn.X1     := 34;

 Rgn.Y1     := 929;

 Rgn.Width  := 200;

 Rgn.Height := 200;


 Fit.BitmapRead(Bmp, Rgn);

 Bmp.SaveToFile('bitmap.bmp');


 // free

 Bmp.Free;

 Fit.Stream.Free;

 Fit.Free;


end;


Output

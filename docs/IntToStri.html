function IntToStri(const Value: Int64; Sign: Boolean; Prec: Word): string;

overload;


Description:


The function converts Integer into String, by the standard.


The function is overloaded:


  1. see the function IntToStr;


  1. Sign - display a sign Value;
    Fmt - format of the string, see the function Format;


  1. Sign - display a sign Value;
    Prec - the number of characters for text presentation of Value.


Code Sample:


var

 S1, S2, S3, S4, S5: string;

begin

 // Simple

 S1 := IntToStri(123);

 // Sign

 S2 := IntToStri(+123, True, '%d');

 S3 := IntToStri(-123, False, '%d');

 // Prec

 S4 := IntToStri(123, False, 3);

 S5 := IntToStri(123, False, 4);

end;


Output


// Simple


S1 = '123'


// Sign


S2 = '+123'

S3 = '-123'


// Prec


S4 = '123'

S5 = '0123'

An example of using the constants cCastBoolean


1. We will read the line of header as Boolean


Input



00

header fragment

SIMPLE  =                    T / conforms to FITS standard                      


var

 Fit: TFitsFileFrame;

 Index: Integer;

 Item: TLineItem;

 V: Boolean;

begin

 // open file for read

 Fit := TFitsFileFrame.CreateJoin('some.fit', cFileRead);

 // search and read for line with a keyword 'SIMPLE'

 Index := Fit.LineBuilder.IndexOf('SIMPLE');

 if Index >= 0 then

 begin

   Item := Fit.LineBuilder.Items[Index, cCastBoolean];

   V    := Item.Value.Bol;

 end;

 // free

 Fit.Free;

end;


Output


V = True


2. We will add the line of header as Boolean


var

 Fit: TFitsFileFrame;

 Item: TLineItem;

begin

 // open file for read and write

 Fit := TFitsFileFrame.CreateJoin('some.fit', cFileReadWrite);

 // build item

 Item.Keyword       := 'SOMEKEY';

 Item.ValueIndicate := True;

 Item.Value.Bol     := False;

 Item.NoteIndicate  := True;

 Item.Note          := 'this is my value of boolean type';

 // add item in header as cCastBoolean

 Fit.LineBuilder.Add(Item, cCastBoolean);

 // free

 Fit.Free;

end;


Output



12

header fragment

SOMEKEY =                    F / this is my value of boolean type

Error Codes

This topic contains all the codes of exceptions that generates DeLaFits


1

ERROR_SYS

An error in the system functions


1000

ERROR_FITS

Basic constant for the error codes of DeLaFits.

All exceptions of library have large values of codes

1001

ERROR_NULL

The error code is not defined, this value of Code by default

1002

ERROR_UNKNOWN

Unknown error

1003

ERROR_HDUCORE_CONVERT

The error code of formation a 2D-core header.

The specified dimension is not equal to two or one dimension is equal to zero, see HduCoreMultTo2d


2000

ERROR_STRING

Basic constant for the error codes of conversion StringToValue and ValueToString

2001

ERROR_STRING_LENGTH

Error of converting String into TLineItem.

The length of the input string less than 80 characters

2002

ERROR_STRING_BOOLEAN

Error of converting String into Boolean.

The string can contain only values of "T" or "F"

2003

ERROR_STRING_BITPIX

Error of converting String into TBitPix.

The string can contain only values of: "-64", "-32", "8", "16", "32" or "64"

2004

ERROR_STRING_INTEGER

Error of converting String into Integer

2005

ERROR_STRING_FLOAT

Error of converting String into Float

2006

ERROR_STRING_PARSE

Error of separation line into parts (packages): <keyword-separator-value-separator-note>

2007

ERROR_STRING_DATETIME

Error of converting String into TDateTime

2008

ERROR_STRING_COORD

Error of converting String into Coordinate (Right ascension or Declination).

The error occurs if DeLaFits can not recognize the string "HH: MM: SS. (S)" or

"± DD: MM: SS. (S)"

2009

ERROR_STRING_RA_VALUE

Error of converting of Right ascension α:

  • 00.0 ≤ α ≤ 24.0 in hour;
  • 000.0 ≤ α ≤ 360.0 in degrees;

2010

ERROR_STRING_DE_VALUE

Error of converting of Declination δ:

  • -90.0 ≤ δ ≤ +90.0 in degrees;


3000

ERROR_CLASSES

The basic constant of the codes of errors

3001

ERROR_STREAM_NOT_ASSIGNED

Stream not assigned.

In the constructor of object (read/write) is entered a stream is nil

3002

ERROR_STREAM_LOW_SIZE

Invalid size of the stream.

The minimum stream size 2880 bytes

3003

ERROR_STREAM_INVALID

Invalid stream header.

Not found keyword SIMPLE or END

3004

ERROR_STREAM_READ

Error of reading the header of stream.

Access to the stream is locked or incorrect size a readable block

3005

ERROR_STREAM_WRITE

Error of writing into header of the stream.

Access to the stream is locked or incorrect size a rewritable block

3006

ERROR_CORE_INVALID

Incorrect core of header of file

3007

ERROR_CORE_TWO_INVALID

Incorrect 2D-core of header of file

3008

ERROR_STREAM_TWO_LOW_SIZE

Invalid size of the stream:

  • header size must be proportional to 2880 bytes;
  • data block size of no less than (NAXIS1 x NAXIS2 x BitPixByteSize(BITPIX)) bytes;

3101

ERROR_COMPOSE_UNKNOWN_CAST

Unknown rule (constant) for building the line out of TLineItem in the methods of the class TLineBuilder

3102

ERROR_PARSE_UNKNOWN_CAST

Unknown rule (constant) for convert the line into TLineItem in the methods of the class TLineBuilder

3200

ERROR_LINEINSERT_INDEX

Incorrect a index for the inserting a new line into the header.

The index must not be greater than the index of keyword END, see LineCount

3201

ERROR_LINEINSERT_KEYWORD

Incorrect value of keyword when inserting a new line into header of file.

The keyword in a line must be uppercase

3202

ERROR_LINEINSERT_CORE_KEYWORD

Incorrect value of keyword when inserting a new line into header of file.

The new line contains the keyword that defines core of: SIMPLE, BITPIX, NAXIS, NAXISn, BSCALE or BZERO.

HduCoreAuditing := False disables a checking this condition

3203

ERROR_LINEINSERT_CORE_BREAK

Attempting to insert a new line between lines of the header that define the core.

HduCoreAuditing := False disables a checking this condition

3204

ERROR_LINEINSERT_CORE_INDEX

Attempting to insert a new line in the position 0.

HduCoreAuditing := False disables a checking this condition

3205

ERROR_LINEDELETE_INDEX

Incorrect value of the index when removing header line.

Index ∈ (0; LineCount - 1)

3206

ERROR_LINEDELETE_COUNT

Incorrect value of parameter Count1 when removing header lines.

Block of lines outside the range (0; LineCount - 1]

3207

ERROR_LINEDELETE_CORE

Header line, which will be removed, contains a keyword defining the core: SIMPLE, BITPIX, NAXIS, NAXISn, BSCALE or BZERO.

HduCoreAuditing := False disables a checking this condition

3208

ERROR_LINEEXCHANGE_INDEX

Incorrect value of indexes when exchange a lines of the header.

Index ∈ (0; LineCount - 1)

3209

ERROR_LINEEXCHANGE_CORE

Attempting to exchange the lines that define the core, i.e. contain keywords SIMPLE, BITPIX, NAXIS, NAXISn.

HduCoreAuditing := False disables a checking this condition

3210

ERROR_LINEMOVE_INDEX

Incorrect value of indexes when moving a lines of the header.

Index ∈ (0; LineCount - 1)

3211

ERROR_LINEMOVE_CORE

Incorrect value of indexes when moving a lines of the header.

The process of moving affects at the line, that define the core

3212

ERROR_LINEREAD_INDEX

Incorrect value of index when reading a lines of the header.

Index ∈ [0; LineCount - 1]

3213

ERROR_LINEREAD_COUNT

Incorrect number of lines to read.

When reading a block of lines (several pieces), its size should be in the range [0, LineCount - 1]

3214

ERROR_LINEWRITE_INDEX

Incorrect value of index when writing a lines of the header.

Index ∈ [0; LineCount - 1]

3215

ERROR_LINEWRITE_COUNT

Incorrect number of lines to write.

When writing a block of lines (several pieces), its size should be in the range [0, LineCount - 1]

3216

ERROR_LINEWRITE_KEYWORD

Incorrect value of the keyword when writing new line.

The keyword in a line must be uppercase

3217

ERROR_LINEWRITE_CORE_KEYWORD

Attempting to rewrite line that specifies the core.

HduCoreAuditing := False disables a checking this condition

3218

ERROR_LINEWRITE_CORE_INDEX

Attempting to rewrite line at index 0.

HduCoreAuditing := False disables a checking this condition

3219

ERROR_LINEWRITE_CORE_BREAK

Attempting to rewrite a line that specifies the core.

HduCoreAuditing := False disables a checking this condition

3220

ERROR_GETLINESKEYWORD_INDEX

Incorrect value of index when reading a keyword.

3221

ERROR_SETLINESKEYWORD_INDEX

Incorrect value of index when writing a keyword.

3222

ERROR_SETLINESKEYWORD_CORE_KEYWORD

Attempting to write/rewrite the keyword defining the core: SIMPLE, BITPIX, NAXIS, NAXISn, BSCALE, BZERO.

HduCoreAuditing := False disables a checking this condition

3223

ERROR_SETLINESKEYWORD_CORE_INDEX

Attempting to rewrite keyword at index 0.

HduCoreAuditing := False disables a checking this condition

3300

ERRROR_DATAACCESS_BITPIX_UNKNOWN

Uncertain (unknown) BitPix

3301

ERRROR_DATAACCESS_REP_UNKNOWN

Uncertain (unknown) Rep

3302

ERRROR_DATAACCESS_PROC_NIL

Undefined the functions of data access: read/write

Check the project conditional directives

3303

ERRROR_DATAACCESS_RGN_INVALID

Incorrectly specified region of data access.

The region must be in the full data-region of frame

3304

ERRROR_DATAACCESS_PIXEL_INVALID

Incorrect coordinates of the pixel when accessing data.

The pixel (X, Y) must be in the full data-region of frame

3305

ERRROR_DATAACCESS_DATA_NIL

Undefined the buffer of user (Data = nil) when accessing data: read/write

3306

ERRROR_DATAACCESS_DATA_COLSCOUNT

Incorrectly specified region of user buffer.

Size (number of columns) of user buffer is not match the specified region

3307

ERRROR_DATAACCESS_DATA_ROWSCOUNT

Incorrectly specified region of user buffer.

Size (number of rows) of user buffer is not match the specified region


4000

ERROR_MATH

The basic code a errors for unit DeLaFitsMath: mathematics

4001

ERROR_MATRIX_ZTERM

Violation of the affine properties of matrix. Z-term different from (0, 0, 1):

M11

M21

M31 ≡ 0

M12

M22

M32 ≡ 0

M13

M23

M33 ≡ 1

In matrix of affine transformations must always be Z-term, that is equal

0

0

1

See TGraphicGeom

4002

ERROR_MATRIX_DET_ZERO

Violation of the affine properties of matrix. The determinant of matrix is equal to the zero.

The determinant of the matrix of affine transformations must always be is not equal to the zero.

See TGraphicGeom


5000

ERROR_GRAPHICS

The basic code a errors for unit DeLaFitsGraphics: the graphic image of the data

5001

ERROR_SCALING_INVALID

Invalid value of scaling.

The scale factor is must not be zero

5002

ERROR_SHEAR_INVALID

Invalid value of shear.

Shear angle ∈ [-90°; +90°]

5101

ERROR_HISTOGRAMDYNAMICRANGE_INVALID

Incorrect values for the indexes of dynamic range of the histogram.

Index1 <= Index2

5102

ERROR_BRIGHTNESS_INVALID

Incorrect values for brightness of graphic image of the data.

Brightness ∈ [-1.0; +1.0]

5103

ERROR_CONTRAST_INVALID

Incorrect values for contrast of graphic image of the data.

Contrast ∈ [0.0; 3.0]

5104

ERROR_GAMMA_INVALID

Incorrect values for gamma of graphic image of the data.

Gamma ∈ [0.0; +3.0]

5105

ERROR_PALETTE_UNKNOWN

Incorrect values for palette of graphic image of the data.

Palette ≠ nil

5201

ERRROR_BITMAP_NIL

A pointer to TBimap object is nil, see TFitsBitmap.BitmapRead

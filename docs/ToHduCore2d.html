Type: function


function ToHduCore2d(Simple: Boolean; BitPix: TBitPix; NAxis1, NAxis2: Word;

BScale: Double = 1.0; BZero: Double = 0.0): THduCore2d;


Description:


The function creates a record THduCore2d.

Input parameters is not checked.


Code Sample:


var

 R: THduCore2d;

begin

 R := ToHduCore2d(True, bi32c, 100, 200);

end;


Output


R.Simple    = True;

R.BitPix    = bi32c;

R.NAxis1 = 100;

R.NAxis2 = 200;

R.BZero     = 0.0;

R.BScale    = 1.0;

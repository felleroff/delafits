Example cCastRaDe

Const ›› Cast ››

An example of using the constants cCastRa, cCastDe


Default:

  • in the header (text representation): Ra ∈ [0 .. 24h], De ∈ [-90° ... + 90°];
  • in the TLineBuilder (computing): Ra ∈ [0° ... 360°], De ∈ [-90° ... +90°];


To change the parameters, see property of TLineBuilder.Fmt: PFormatLine.


1. We will read the line of header as Ra and De


Input



10

11

header fragment

RA      = '22 27 58.07'        / [hms J2000] Target right ascension            

DEC     = '-11 40 26.8'        / [dms +N J2000] Target declination              


var

 Fit: TFitsFileFrame;

 Index: Integer;

 Item: TLineItem;

 Ra, De: Extended;

begin

 // open file for read

 Fit := TFitsFileFrame.CreateJoin('some.fit', cFileRead);

 // search and read for line with a keyword 'RA'

 Index := Fit.LineBuilder.IndexOf('RA');

 if Index >= 0 then

 begin

   Item := Fit.LineBuilder.Items[Index, cCastRa];

   Ra   := Item.Value.Ext;

 end;

 // search and read for line with a keyword 'DEC'

 Index := Fit.LineBuilder.IndexOf('DEC');

 if Index >= 0 then

 begin

   Item := Fit.LineBuilder.Items[Index, cCastDe];

   De   := Item.Value.Ext;

 end;

 // free

 Fit.Free;

end;


Output


Ra = 336.99195833  // in degree [0° ... 360°]

De = -11.674111111 // in degree [-90° ... +90°]


2. We will add the line of header as Ra and De


var

 Fit: TFitsFileFrame;

 Item: TLineItem;

begin

 // open file for read and write

 Fit := TFitsFileFrame.CreateJoin('some.fit', cFileReadWrite);

 // build and add item as Ra

 Item.Keyword       := 'RA';

 Item.ValueIndicate := True;

 Item.Value.Ext     := 336.99195833;

 Item.NoteIndicate  := True;

 Item.Note          := 'this is my value of right ascension';

 Fit.LineBuilder.Add(Item, cCastRa);

 // build and add item as De

 Item.Keyword       := 'DEC';

 Item.ValueIndicate := True;

 Item.Value.Ext     := -11.674111111;

 Item.NoteIndicate  := True;

 Item.Note          := 'this is my value of declination';

 Fit.LineBuilder.Add(Item, cCastDe);

 // free

 Fit.Free;

end;


Output



10

11

header fragment

RA      = '22:27:58.07'        / this is my value of right ascension            

DEC     = '-11:40:26.8'        / this is my value of declination

function PosChar(Ch: Char; const S: string; StartPos: Integer = 1): Integer;


Description:


Function is looking for the index of symbol Ch in the string S.


StartPos sets the index of the beginning of the search.

Search - a sequential scan of characters of line S from left to right.


The function is case sensitive.


If successful, the function returns the index of the symbol C in a string S, if the symbol C is not found the function returns 0.


Code Sample:


var

 I1, I2, I3, I4: Integer;

begin

 I1 := PosChar('x', 'xABxC', 1);

 I2 := PosChar('X', 'xABxC', 1);

 I3 := PosChar('x', 'xABxC', 2);

end;


Output


I1 = 1

I2 = 0

I3 = 4

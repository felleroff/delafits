Unit: DeLaFitsString


Type: function


function StriToRa(const Value: string; Tip: TCoordMeasur = cmHour): Extended;


Description:


The function converts String into Right ascension, by the standard.


The function returns the value of Right ascension strictly in degrees: [0° .. 360°).


Tip - the format of Value: degree or hour. The function automatically detects the format, Tip - a hint.


For incorrect Value the function generates an exceptions:

2005:ERROR_STRING_FLOAT;

2008:ERROR_STRING_COORD;

2009:ERROR_STRING_RA_VALUE;


Code Sample:


var

 R1, R2, R3, R4: Extended;

begin

 R1 := StriToRa('180.1234');

 R2 := StriToRa('12.00823');

 R3 := StriToRa('180:07:24.2');

 R4 := StriToRa('12:00:29.62');

end;


Output


R1 = 180.1234

R2 = 180.1234

R3 = 180.1234

R4 = 180.1234

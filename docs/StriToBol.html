function StriToBol(const Value: string): Boolean;


Description:


The function converts String into Boolean, by the standard.

The function is case sensitive.


For incorrect Value the function generates an exception 2002:ERROR_STRING_BOOLEAN.


Code Sample:


var

 B1, B2: Boolean;

begin

 B1 := StriToBol('T');

 B2 := StriToBol('F');

end;


Output


B1 = True

B2 = False

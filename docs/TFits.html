TFits

Classes ››

Unit: DeLaFitsClasses


Type: class


Inherited from: System.TObject


Description:


TFits is the base class for all classes DeLaFits.

TFits defines a constructor to create an object − new or stream-based.

TFits contains methods for working with TStream.


Protected fields

Core of header of frame

An array of all the keywords in the header of frame

FStream

The stream − class TStream − direct access to the FITS-stream


Protected methods


CheckCore virtual

Validates the core of stream on conformity to standard. Method is called in the class constructor

CheckStream virtual

Validates the stream on conformity to standard. Method is called in the class constructor

Init virtual

Initializes the fields of the class. Method is called in the class constructor, after checks

StreamFill

Fills stream content

StreamRead

Read a block of data of stream into a the buffer TStreamBuffer

StreamShift

Overwrites the content of the stream:

Shift > 0 − content moves down;

Shift < 0 − content moves up;

StreamWrite

Write data of buffer TStreamBuffer into a stream

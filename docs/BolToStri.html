Unit: DeLaFitsString


Type: function


function BolToStri(const Value: Boolean): string;


Description:


The function converts Boolean into String, by the standard.


Code Sample:


var

 S1, S2: string;

begin

 S1 := BolToStri(True);

 S2 := BolToStri(False);

end;


Output


S1 = 'T'

S2 = 'F'

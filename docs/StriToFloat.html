function StriToFloat(const Value: string): Extended;


Description:


The function converts String into Float, by the standard.

For incorrect Value the function generates an exception 2005:ERROR_STRING_FLOAT.


Code Sample:


var

 V1, V2: Extended;

begin

 V1 := StriToFloat('123.45');

 V2 := StriToFloat('123,45');

end;


Output


V1 = 123.45

V2 = 123.45

property HduCore: THduCore read FHduCore;


Description:


The core of header. TFitsHeader creates an entry THduCore, which describes the basic properties of the stream: SIMPLE, BITPIX, the dimension of the data block NAXIS, an array of values NAXISn, coefficients BSCALE and BZERO.

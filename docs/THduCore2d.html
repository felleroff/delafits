THduCore2d

Types ››

Unit: DeLaFitsCommon


Type: type


THduCore2d = record

 Simple: Boolean;

 BitPix: TBitPix;

 NAxis1, NAxis2: Word;

 BScale: Double;

 BZero: Double;

end;


Description:


Main meta information a two-dimensional file: values of basic keywords of header. These values describe data: physical representation, dimensionality, bit depth. Basic keywords: SIMPLE, BITPIX, NAXIS, NAXIS1, NAXIS2, BSCALE and BZERO. See a detailed description of Keywords defined in the FITS Standard.


Simple: Boolean;


Does file conform to the Standard? True - yes, False - no. DeLaFits support access to the data only for Simple files;


BitPix: TBitPix;


Bits per data value, see TBitPix;


NAxis1, NAxis2: Word;


The length of two-dimensional array. According to Standard (version 3.0, page 14-15) two-dimensional pixel data in the file are arranged as follows (example for frame Naxis1 = 7; Naxis2 = 8):



Access to the pixels of two-dimensional data array:


Pix := Data[Xmax, Ymax]

Pix := Data[ColsCount - 1, RowsCount - 1]

Pix := Data[NAXIS1 - 1, NAXIS2 - 1]


BScale, BZero: Double;


Linear factor and zero point in scaling equation: physical_value = BZERO + BSCALE * array_value.

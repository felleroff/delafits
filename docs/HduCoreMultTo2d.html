Unit: DeLaFitsCommon


Type: function


function HduCoreMultTo2d(const Core: THduCore): THduCore2d;


Description:


The function converts THduCore (core of header) into THduCore2d (the two-dimensional core of header).

Core.NAxis must be equal to 2.

For incorrect Core the function generates an exception 1003:ERROR_HDUCORE_CONVERT.


Code Sample:


1. A simple example, the correct parameters:


var

 HduCore: THduCore;

 HduCore2d: THduCore2d;

begin

 HduCore.Simple    := True;

 HduCore.BitPix    := bi16c;

 HduCore.NAxis     := 2;

 SetLength(HduCore.NAxisn, HduCore.NAxis); // = 2

 HduCore.NAxisn[0] := 100;

 HduCore.NAxisn[1] := 200;

 HduCore.BScale    := 1.0;

 HduCore.BZero     := 0.0;

 HduCore2d := HduCoreMultTo2d(HduCore);

end;


Output:


HduCore2d.Simple := True;

HduCore2d.BitPix := bi16c;

HduCore2d.NAxis1 := 100;

HduCore2d.NAxis2 := 200;

HduCore2d.BScale := 1.0;

HduCore2d.BZero  := 0.0;


2. Incorrect parameters, exception:


var

 HduCore: THduCore;

 HduCore2d: THduCore2d;

begin

 HduCore.Simple    := True;

 HduCore.BitPix    := bi16c;

 HduCore.NAxis     := 1;

 SetLength(HduCore.NAxisn, HduCore.NAxis); // = 1! error

 HduCore.NAxisn[0] := 100;

 HduCore.BScale    := 1.0;

 HduCore.BZero     := 0.0;

 HduCore2d := HduCoreMultTo2d(HduCore);

end;


Output:


Exception class EFitsException with message

'DeLaFits.EFitsException

Invalid value NAxis(n)

Code 1003'

TBand

Types ››

Unit: DeLaFitsCommon


Type: type


TBand = array [0 .. cSizeBand - 1] of Extended; // cSizeBand = 256;


Description:


Relationship between the physical value of the pixel of data and the RGB-color of the pixel of graphic image of data. The size of the Band is 256 items. The items of Band is values of the pixels of data of the frame.

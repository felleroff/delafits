procedure GraphicPrepare(var Pixels: TPixels; const Rgn: TRgn);


Description:


The method allocates memory for two-dimensional array of indexes of the palette. The analogue of the SetLength procedure.


  • Pixels - the graphic image of the data, see TPixels;
  • Rgn - the region of the graphic image of the data, size of array Pixels;


Code Sample:


var

 Fit: TFitsGraphic;

 Rgn: TRgn;

 Pixels: TPixels; // ~array of array of Byte;

 PixelsWidth, PixelsHeight: Integer;

begin

 // open file for read

 Fit := TFitsGraphic.CreateJoin(TFileStream.Create('some.fit', cFileRead));

 // define region

 Rgn.X1     := 0;

 Rgn.Y1     := 0;

 Rgn.Width  := 10;

 Rgn.Height := 20;

 // prepare Pixels array, ~SetLength(Pixels, Rgn.Width, Rgn.Height)

 Fit.GraphicPrepare(Pixels, Rgn);

 // Get size of Pixels array

 PixelsWidth  := Length(Pixels);

 PixelsHeight := Length(Pixels[0]);

 // free

 Pixels := nil;

 Fit.Stream.Free;

 Fit.Free;

end;


Output


PixelsWidth  = 10;

PixelsHeight = 20;

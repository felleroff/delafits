FHduKeywords

Classes ›› TFits ››

Unit: DeLaFitsClasses


Type: field


Visibility: protected


Member Of: TFits


FHduKeywords: THduKeywords;


Description:


An array of all the keywords in the header of frame. The array contains all the keywords of the frame, including END and the following blank lines. Count of elements in the array ~36.


Numbering of lines a header begins with zero.

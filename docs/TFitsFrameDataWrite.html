procedure DataWrite(Data: Pointer); overload;


procedure DataWrite(Data: Pointer; const Rgn: TRgn); overload;


Description:


This method writes a data block.


The size and coordinates of block sets parameter Rgn. If this parameter is not specified (overload), method will write the full region of data.


Data is a pointer to a two-dimensional array, the user buffer. After execution of the method, data block of frame will contain the Data.


Type of array elements Data (type of numbers) the user specifies in the DataRep property.



Code Sample:


var

 Fit: TFitsFrame;

 Rgn: TRgn;

 Data32c: array of array of Integer;

 X, Y: Integer;

begin

 // open file for read and write

 Fit := TFitsFrame.CreateJoin(TFileStream.Create('some.fit', cFileReadWrite));

 // define region

 Rgn.X1 := 0;

 Rgn.Y1 := 0;

 Rgn.ColsCount := 4;

 Rgn.RowsCount := 3;

 // set the elements of user buffer: array of array of Integer

 Fit.DataRep := rep32c;

 // prepare user buffer

 Fit.DataPrepare(Pointer(Data32c), Rgn);

 for X := 0 to Rgn.ColsCount - 1 do

 for Y := 0 to Rgn.RowsCount - 1 do

   Data32c[X, Y] := Random(1000);

 // write the user buffer in the frame

 Fit.DataWrite(Pointer(Data32c), Rgn);

 // free

 Fit.Stream.Free;

 Fit.Free;

 Data32c := nil;

end;

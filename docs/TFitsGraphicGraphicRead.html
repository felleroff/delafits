procedure GraphicRead(out Palette: TPalette; var Pixels: TPixels;

   const Rgn: TRgn); virtual;


Description:


The method of creating the graphic image of the data.


The Graphic image of the data it is two-dimensional array of indices of the palette. For each Value of pixel of data is assigned a color - the element of palette. Thus, Pixels is a two dimensional array of pixels of data that is converted to color (elements of palette).


Such an array is easy to convert into a known graphic format, for example, BMP, see TFitsBitmap.


  • Palette - the palette of graphic image of the data, see TPalette;
  • Pixels - the graphic image of the data, see TPixels;
  • Rgn - the region of the graphic image of the data, size of array Pixels;

var

 Fit: TFitsGraphic;

 Rgn: TRgn;

 Palette: TPalette; // ~array [0 .. 255] of (R,G,B)

 Pixels: TPixels;   // ~array of array of Byte;

begin


 // open file for read

 Fit := TFitsGraphic.CreateJoin(TFileStream.Create('some.fit', cFileRead));


 // define region

 Rgn.X1     := 34;

 Rgn.Y1     := 929;

 Rgn.Width  := 300;

 Rgn.Height := 250;


 // prepare Pixels array, ~SetLength(Pixels, Rgn.Width, Rgn.Height)

 Fit.GraphicPrepare(Pixels, Rgn);


 // Get graphic image of the data

 Fit.GraphicRead(Palette, Pixels, Rgn);


 // ... some code: convert Palette and Pixels into the known graphic format

 // and get the image of the region of the frame covert


 // free

 Pixels := nil;

 Fit.Stream.Free;

 Fit.Free;


end;

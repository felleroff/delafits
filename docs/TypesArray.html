Description:


One- and two-dimensional arrays of standard numerical data types. Type is used in the methods of read/write data.


DeLaFits does not obligate user to use these types (arrays). Methods of data access uses the standard Pointer type, i.e. it is possible to use simple coercion of types, for example:


var

 Data: array of array of Integer;

begin

...

 Fits.DataRead(Pointer(Data));

...

end;

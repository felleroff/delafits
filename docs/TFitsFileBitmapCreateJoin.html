constructor CreateJoin(const AFileName: string; AFileMode: Word);


Description:


Creates an object from an existing FITS-file.

The FITS-file must be two-dimensional: NAXIS = 2.


  • AFileName − the name of an existing file in the format FITS;
  • AFileMode − mode a opening of file − read, read and write. See description cFileRead and cFileReadWrite in topic Const;


TFitsFileBitmap itself creates and destroys a file stream − Stream: TFileStream.


Code Sample:


var

 Fit: TFitsFileBitmap;

begin

 Fit := TFitsFileBitmap.CreateJoin('some.fit', cFileReadWrite);

 // ... some code

 Fit.Free;

end;

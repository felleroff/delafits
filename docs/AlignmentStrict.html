Unit: DeLaFitsString


Type: function


function AlignmentStrict(const S: string; Width: Integer): string;


Description:


The function returns a new string S.

The length of the new string = Width.

The sign Width determines the place of additions: at the beginning or at the end of the line S.


  • |Width| <= Length(S) - set the length of the string in Width, truncate;
  • Width < 0 - addition at the end of S;
  • Width > 0 - addition at the beginning of S;


Code Sample:


var

 S1, S2, S3, S4, S5: string;

begin

 S1 := AlignmentStrict('text', -5);

 S2 := AlignmentStrict('text', -3);

 S3 := AlignmentStrict('text',  0);

 S4 := AlignmentStrict('text', +3);

 S5 := AlignmentStrict('text', +5);

end;


Output


S1 = 'text '

S2 = 'tex'

S3 = ''

S4 = 'tex'

S5 = ' text'

function Quoted(const S: string; Width: Integer = 0): string;


Description:


The function converts String into String, by the standard: quoted a string S.


"A character string value shall be composed only of the set of restricted ASCII text characters, enclosed by single quote characters. A single quote is represented within a string as two successive single quotes".


Width defines length of the string S inside the quotes.


Code Sample:


var

 S1, S2, S3, S4: string;

begin

 S1 := Quoted('text', -5);

 S2 := Quoted('text', ±4);

 S3 := Quoted('text', +5);

 S4 := Quoted('d’text', 0);

end;


Output


S1 = '’text ’'

S2 = '’text’'

S3 = '’ text’'

S4 = '’d’’text’'

function FloatToStri(const Value: Extended; Sign: Boolean; PrecInt, PrecFloat:

Word): string; overload;


Description:


The function converts Float into String, by the standard.


The function is overloaded:


  1. see the function FloatToStr;


  1. Sign - display a sign Value;
    Fmt - format of the string, see the function Format;


  1. Sign - display a sign Value;
    PrecInt - the number of characters of the integer part of Value;
    PrecFloat - the number of characters of the fractional part of Value.


Code Sample:


var

 S1, S2, S3, S4, S5, S6: string;

begin  

 // Simple

 S1 := FloatToStri(123.45);

 // Sign

 S2 := FloatToStri(-123.45, False, '.%2f');

 S3 := FloatToStri(+123.45, True, '.%2f');

 // Rounding

 S4 := FloatToStri(123.45, False, '.%3f');

 S5 := FloatToStri(123.45, False, '.%2f');

 S6 := FloatToStri(123.45, False, '.%1f');

 // Prec

 S7 := FloatToStri(123.45, False, 4, 4);

end;


Output


// Simple


S1 = '123.45'


// Sign


S2 = '-123.45'

S3 = '+123.45'


// Rounding


S4 = '123.450'

S5 = '123.45'

S6 = '123.5'


// Prec


S6 = '0123.4500'

Type: function


function Alignment(const S: string; Width: Integer): string;


Description:


The function returns the aligned line S: complements the blanks for line S.

The sign Width determines the place of additions: at the beginning or at the end of the line S.


  • |Width| <= Length(S) - no additions;
  • Width < 0 - addition at the end of S;
  • Width > 0 - addition at the beginning of S;


Code Sample:


var

 S1, S2, S3: string;

begin

 S1 := Alignment('text', -5);

 S2 := Alignment('text', ±4);

 S3 := Alignment('text', +5);

end;


Output


S1 = 'text '

S2 = 'text'

S3 = ' text'

THistogram

Types ››

Unit: DeLaFitsCommon


Type: type


PHistogramItem = ^THistogramItem;

THistogramItem = record

 Value: Extended;

 Count: Int64;

end;


PHistogram = ^THistogram;

THistogram = array [0 .. MaxListSize - 1] of PHistogramItem;


Description:


Histogram of physical values of the pixels of data of the frame. DeLaFits sets the maximum size for histogram at 50.000 items.


  • THistogramItem.Value - individual value of data;
  • THistogramItem.Count - the number of Value-pixels in the data array;

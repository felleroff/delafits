Type: function


function DeToStri(const Value: Extended; Prec: Word; Notate: TCoordNotate):

   string;


Description:


The function converts Declination into String, by the standard.


  • Value - Declination, in degrees from [-90° .. +90°];
  • Prec - precision of rounding;
  • Notate - the format of the text representation: decimal or hexadecimal;


For incorrect Value the function generates an exception 2010:ERROR_STRING_DE_VALUE.


Code Sample:


var

 S1, S2: string;

begin

 S1 := DeToStri(45.1234, 4, cmDecimal);

 S2 := DeToStri(45.2096, 4, cmSexagesimal);

end;


Output


S1 = '+45.1234'

S2 = '+45:12:34.6'

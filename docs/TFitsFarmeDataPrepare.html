procedure DataPrepare(out Data: Pointer); overload;


procedure DataPrepare(out Data: Pointer; const Rgn: TRgn); overload;


Description:


The method prepares the user buffer: allocates memory for two-dimensional array according to the specified region. Type of elements of the user buffer sets the property DataRep.


In fact, the method invokes the SetLength function.


Method is overloaded. If do not specify the region, then for the user buffer Data will be set by size of the full data region of frame DataRgn.


Code Sample:


var

 Fit: TFitsFrame;

 Rgn: TRgn;

 Data16: array of array of Word;

 Data32: array of array of Integer;

 Cols16, Rows16, Cols32, Rows32: Integer;

begin

 // open file for read and write

 Fit := TFitsFrame.CreateJoin(TFileStream.Create('some.fit', cFileReadWrite));

 // define region

 Rgn.X1 := 0;

 Rgn.Y1 := 0;

 Rgn.ColsCount := 100;

 Rgn.RowsCount := 200;

 // prepare the user buffer of 16-bit: Word

 Fit.DataRep := rep16u;

 Fit.DataPrepare(Pointer(Data16), Rgn);

 Cols16 := Length(Data16);

 Rows16 := Length(Data16[0]);

 // prepare the user buffer of 32-bit: Integer

 Fit.DataRep := rep32c;

 Fit.DataPrepare(Pointer(Data32), Rgn);

 Cols32 := Length(Data32);

 Rows32 := Length(Data32[0]);

 // ... some code

 // free

 Fit.Stream.Free;

 Fit.Free;

 Data16 := nil;

 Data32 := nil;

end;


Output


Cols16 = 100;

Rows16 = 200;


Cols32 = 100;

Rows32 = 200;

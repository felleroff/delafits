property Code: Integer read FCode;


Description:


Code of Exception DeLaFits. The code is set in the constructor a exception. The code identifies the location and nature a exception.


Code Sample:


var

 Code: Integer;

begin

 try

   raise EFitsException.Create('Error Message', 10000);

 except

   on E: EFitsException do

   begin

     Code := E.Code;

   end;

 end;

end;


Output


Code = 1000

An example of using the constants cCastCharsLong


1. We will read from the header a value of long comment as String (one line)


Input



10

11

12

header fragment

COMMENT   Astronomy is one of the few sciences where amateurs can still play an

COMMENT   active role, especially in the discovery and observation of transient

COMMENT   phenomena                                                            


var

 Fit: TFitsFileFrame;

 Index: Integer;

 Item: TLineItem;

 LongComment: string;

begin

 // open file for read

 Fit := TFitsFileFrame.CreateJoin('some.fit', cFileRead);

 // search for line with a keyword 'COMMENT'

 Index := Fit.LineBuilder.IndexOf(cCOMMENT);

 // if found, we read a item

 if Index >= 0 then

 begin

   Item := Fit.LineBuilder.Items[Index, cCastCharsLong];

   LongComment := Item.Value.Str;

   // or used property: LongComment := Fit.LineBuilder.ValuesCharsLong[Index];

 end;

 // free

 Fit.Free;

end;


Output


LongComment = 'Astronomy is one of the few sciences where amateurs can still play an active role, especially in the discovery and observation of transient phenomena';


2. We will add the long comment in the header


var

 Fit: TFitsFileFrame;

 Item: TLineItem;

 LongComment: string;

begin

 // open file for read and write

 Fit := TFitsFileFrame.CreateJoin('some.fit', cFileReadWrite);

 // put long comment

 LongComment :=

     'Astronomy is one of the few sciences where amateurs can still play an ' +

     'active role, especially in the discovery and observation of transient ' +

     'phenomena';

 // build item

 Item.Keyword       := cCOMMENT;

 Item.ValueIndicate := False;

 Item.Value.Str     := LongComment;

 Item.NoteIndicate  := False;

 Item.Note          := '';

 // add item in header as cCastCharsLong

 Fit.LineBuilder.Add(Item, cCastCharsLong);

 // free

 Fit.Free;

end;

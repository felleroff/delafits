Preamble

What is DeLaFits?


DeLaFits − native for Delphi and Lazarus class library for operation with files in the FITS format:

  • building new;
  • reading/editing of the header and of the data;
  • rendering (creation of a graphic image) of the data.


Terminology:

  • The header is a list of records <keyword-separator-value-separator-note>;
  • The data is a two-dimensional array of integers or real numbers;
  • The graphic image of data is a two-dimensional array of indexes of a palette (256 colors). Such an array is easy to convert into a known graphic format, for example, BMP.


DeLaFits uses regions for data access and creation of a graphic image of data.

DeLaFits supports only Single Image FITS − one header and one data unit.

Operations with data and graphics are available only for 2D-files − NAXIS = 2.


Code Samples


uses

 DeLaFitsCommon, DeLaFitsClasses, DeLaFitsGraphics;


{ 1. Build new file }


procedure TForm1.Button1Click(Sender: TObject);

var

 Fit: TFitsFileFrame;

 Hdu: THduCore2d;

begin

 // init header core

 Hdu.Simple := True;

 Hdu.BitPix := bi16c;

 Hdu.NAxis1 := 1024;

 Hdu.NAxis2 := 1024;

 Hdu.BScale := 1.0;

 Hdu.BZero  := 0.0;

 // create new file

 Fit := TFitsFileFrame.CreateMade('some.fit', Hdu);

 // ... some code

 Fit.Free;

end;


{ 2. Read header value }


procedure TForm1.Button2Click(Sender: TObject);

var

 Fit: TFitsFileFrame;

 Index: Integer;

 Ra: Extended;

begin

 // open file for read

 Fit := TFitsFileFrame.CreateJoin('some.fit', cFileRead);

 // search for line with a keyword 'RA'

 Index := Fit.LineBuilder.IndexOf('RA');

 // if found, we read a value

 if Index>= 0 then

 begin

   Ra := Fit.LineBuilder.ValuesRa[Index];

   // ... some code

 end;

 Fit.Free;

end;


{ 3. Edit header value }


procedure TForm1.Button3Click(Sender: TObject);

var

 Fit: TFitsFileFrame;

 Index: Integer;

 Item: TLineItem;

begin

 // open file for read and write

 Fit := TFitsFileFrame.CreateJoin('some.fit', cFileReadWrite);

 // search for line with a keyword 'RA'

 Index := Fit.LineBuilder.IndexOf('RA');

 // if found, we rewrite a value

 if Index >= 0 then

 begin

   Fit.LineBuilder.ValuesRa[Index] := 123.456;

   Fit.LineBuilder.Notes[Index]    := 'This is my value';

 end

 //  if not find, we add a new line

 else

 begin

   // init item

   Item.Keyword       := 'RA';

   Item.ValueIndicate := True;

   Item.Value.Ext     := 123.456;

   Item.NoteIndicate  := True;

   Item.Note          := 'This is my value';

   // add item to header as RA

   Fit.LineBuilder.Add(Item, cCastRa);

 end;

 Fit.Free;

end;


{ 4. Read/write data }


procedure TForm1.Button4Click(Sender: TObject);

var

 Fit: TFitsFileFrame;

 Rgn: TRgn;

 Data: array of array of Integer;

 X, Y: Integer;

begin

 // open file for read and write

 Fit := TFitsFileFrame.CreateJoin('some.fit', cFileReadWrite);

 // init data region

 Rgn.X1        := 0;

 Rgn.Y1        := 0;

 Rgn.ColsCount := 200;

 Rgn.RowsCount := 100;

 // set of data presentation: Integer

 Fit.DataRep := rep32c;

 // prepare data buffer: SetLength(Data, Rgn.ColsCount, Rgn.RowsCount)

 Fit.DataPrepare(Pointer(Data), Rgn);

 // data read

 Fit.DataRead(Pointer(Data), Rgn);

 // ... some code, example

 for Y := 0 to Rgn.RowsCount - 1 do // ~ Length(Data[0])

 for X := 0 to Rgn.ColsCount - 1 do // ~ Length(Data)

   Data[X, Y] := Data[X, Y] + 1;

 // data write

 Fit.DataWrite(Pointer(Data), Rgn);

 // free

 Data := nil;

 Fit.Free;

end;

procedure TForm1.Button6Click(Sender: TObject);

{ 7. Render of some region of the data + affine transformations }


procedure TForm1.Button7Click(Sender: TObject);

var

 Fit: TFitsFileBitmap;

 Bmp: TBitmap;

 Rgn: TRgn;  

begin

 // open file for read

 Fit := TFitsFileBitmap.CreateJoin('some.fit', cFileRead);

 Bmp := TBitmap.Create;

 // init some region

 Rgn.X1     := 0;

 Rgn.Y1     := 0;

 Rgn.Width  := 200;

 Rgn.Height := 200;

 // render data

 Fit.BitmapRead(Bmp, Rgn);

 // ... some code

 Bmp.Free;

 Fit.Free;

end;


General structure DeLaFits


The library is easy to add to your project − DeLaFits a set of modules (pas and inc).


For an user three modules are important:

  • DeLaFitsCommon.pas − the common constants, simple functions;
  • DeLaFitsClasses.pas − access to header and data − building, reading, editing;
  • DeLaFitsGraphics.pas − rendering of the data − creation of a graphic image.


These modules contain the description of classes:

  • TFitsHeader − access to the header;
  • TFitsFrame − access to the data;
  • TFitsBitmap − creation of a graphic image of data in the BMP format.

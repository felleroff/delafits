constructor CreateJoin(AStream: TStream);


Description:


Creates an object from an existing FITS-stream: reading from the stream TStream; parsing the header and creating an array of keywords FHduKeywords.


  • AStream − known data flow in the FITS format. You create and destroy the Stream-object yourself;


Code Sample:


var

 AStream: TFileStream;

 Fit: TFitsHeader;

begin

 AStream := TFileStream.Create('some.fit', fmOpenReadWrite or fmShareDenyNone);

 Fit := TFitsHeader.CreateJoin(AStream);

 // ... some code

 Fit.Free;

 AStream.Free;

end;

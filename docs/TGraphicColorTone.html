procedure Tone(const ABrightness, AContrast, AGamma: Double);


Description:


The method sets the brightness, contrast and gamma of the graphic image of data simultaneously. The method rearranges the Band.


Brightness, contrast and gamma values have a certain range:


  • -1.0 ≥ Brightness ≤ +1.0;
  • 0.0 ≥ Contrast ≤ +3.0;
  • 0.0 ≥ Gamma ≤ +3.0;

Code Sample:


var

 Fit: TFitsGraphic;

 Brightness0, Contrast0, Gamma0: Double;

 Brightness1, Contrast1, Gamma1: Double;


begin


 // open file for read

 Fit := TFitsGraphic.CreateJoin(TFileStream.Create('some.fit', cFileRead));


 // get value of Tone by default

 Brightness0 := Fit.GraphicColor.ToneBrightness;

 Contrast0   := Fit.GraphicColor.ToneContrast;

 Gamma0      := Fit.GraphicColor.ToneGamma;


 // color correction

 Fit.GraphicColor.Tone(0.5, 2.0, 1.5);


 // get new value of Tone

 Brightness1 := Fit.GraphicColor.ToneBrightness;

 Contrast1   := Fit.GraphicColor.ToneContrast;

 Gamma1      := Fit.GraphicColor.ToneGamma;


 // ... some code


 // free

 Fit.Stream.Free;

 Fit.Free;


end;

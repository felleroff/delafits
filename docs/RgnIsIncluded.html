function RgnIsIncluded(const R, Rsub: TRgn): Boolean; overload;


function RgnIsIncluded(const R: TRgn; const Xsub, Ysub: Integer): Boolean;

 overload;


Description:


The function checks:


  • region a Rsub ⊆ region a R;
  • pixel a (Xsub, Ysub) ⊆ region a R;


Code Sample:


var

 R, R1, R2: TRgn;

 X, Y: Integer;

 B1, B2, B3: Boolean;

begin

 R := ToRgnCoRo(0, 0, 10, 20);

 R1 := ToRgnCoRo(5, 5, 5, 15);

 R2 := ToRgnCoRo(5, 5, 5, 16);

 X := 9;

 Y := 19;

 B1 := RgnIsIncluded(R, R1);

 B2 := RgnIsIncluded(R, R2);

 B3 := RgnIsIncluded(R, X, Y);

end;


Output


B1 = True;

B2 = False;

B3 = True;

TCoordMeasur

Types ››

Unit: DeLaFitsCommon


Type: type


TCoordMeasur = (cmDegree, cmHour);


Description:


Text representation of the coordinates (Right ascension α): degree or hour.


  • cmDegree - degree: DDD:MM:SS.(s), e.g. 123°45'56".7;


  • cmHour - hour: HH:MM:SS.(s), e.g. 12h23m45.6s;

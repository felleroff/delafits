property ValuesCharsLong[Index: Integer]: string Index cCastCharsLong

   read GetValuesStr write SetValuesStr;


Description:


List of the Values from the lines of header as Long Chars-value, "as is". This property may be useful when reading values of line with a keyword COMMENT and HISTORY.


Line - is a record from of items <keyword-separator-value-separator-note>, see TLineItem.


TLineBuilder.Count returns the number of significant lines of header (including END).


Numbering of lines a header begins with zero

header fragment

COMMENT   Astronomy is one of the few sciences where amateurs can still play an

var

 Fit: TFitsHeader;

 Str: string;

begin

 Fit := TFitsHeader.CreateJoin(TFileStream.Create('some.fit', cFileRead));

 Str := Fit.LineBuilder.ValuesCharsLong[10];

 Fit.Stream.Free;

 Fit.Free;

end;

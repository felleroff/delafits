TPalette

Types ››

Unit: DeLaFitsCommon


Type: type


TPaletteItem = record

 R, G, B: Byte;

end;


PPalette = ^TPalette;

TPalette = array [0 .. cSizePalette - 1] of TPaletteItem; // cSizePalette = 256


Description:


The palette of graphical image of data. The size of the palette is 256 items. The items of palette is RGB-color.

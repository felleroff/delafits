function Clr: TGraphicGeom;


Description:


The method sets the matrix of affine transformations to the original state: the coordinate system of the frame is identical to the coordinate system of the scene, see Coordinate system.


The method returns a reference to TGraphicGeom, to be able to build a structure:


Fit.GraphicGeom.Clr.Rot(Angle, xy00).Scl(sX, sY, xy00) etc.


Code Sample:


var

 Fit: TFitsGraphic;

 M0, M1, M2: TMatrix; // ~array [1 .. 3, 1 .. 3] of Double;


begin


 // open file for read

 Fit := TFitsGraphic.CreateJoin(TFileStream.Create('some.fit', cFileRead));


 // Matrix of affine transformations: the initial state

 M0 := Fit.GraphicGeom.MatrixFrm;


 // Matrix of affine transformations: after scaling

 Fit.GraphicGeom.Scl(2.5, 2.5, xy00);

 M1 := Fit.GraphicGeom.MatrixFrm;


 // Matrix of affine transformations: returns the initial state

 Fit.GraphicGeom.Clr;

 M2 := Fit.GraphicGeom.MatrixFrm;


 // ... some code


 // free

 Fit.Stream.Free;

 Fit.Free;


end;
